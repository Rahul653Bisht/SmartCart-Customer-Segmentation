# SmartCart Customer Segmentation using KMeans Clustering

##  Project Overview

This project focuses on customer segmentation using the KMeans Clustering algorithm. The goal is to group customers based on their purchasing behavior, income, and demographic information. These customer segments can help businesses create targeted marketing strategies and improve customer engagement.

---

## Objectives

- Clean and preprocess customer data.
- Handle missing values and outliers.
- Perform feature engineering.
- Encode categorical features.
- Scale numerical features.
- Reduce dimensionality using PCA.
- Find the optimal number of clusters using Elbow Method and Silhouette Score.
- Segment customers using KMeans Clustering.
- Analyze and profile each customer segment.

## Workflow

1. Load Dataset
2. Handle Missing Values
3. Feature Engineering
4. Remove Unnecessary Columns
5. Handle Outliers
6. Feature Encoding
7. Feature Scaling
8. PCA for Dimensionality Reduction
9. Find Optimal K (Elbow Method & Silhouette Score)
10. Apply KMeans Clustering
11. Analyze Customer Segments

---

##  Results

The customers were successfully divided into four meaningful clusters based on their purchasing behavior, income, family status, and spending patterns. Cluster profiling was performed to understand the characteristics of each customer segment.
